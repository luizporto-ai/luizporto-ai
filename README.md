# Luiz Porto

Construindo onde **IA agêntica**, **design** e **educação** se cruzam.

Brasileiro. Construtor por compulsão. Acredito que software de verdade nasce de obsessão por detalhe — não de framework do mês.

---

## O que estou construindo

**[Avelum](https://avelum.com.br)** — Escola de IA agêntica. Formação para construtores: sair do prompt e operar sistemas que pensam, decidem e executam. Marca-mãe sob a qual tudo abaixo acontece.

**Total Assistente** — Assistente pessoal multi-agente em produção. Onde aprendo escalabilidade conversacional na prática — auditorias, fixes, testes de carga, repetir.

**Rayo** — Navegador chat-first com Chromium e três modos funcionais (assistente, agente, autônomo). Repensa o ato de "abrir o navegador" do zero. Pivot validado em abr/2026.

**Avelum Labs** — Plataforma educacional da escola. Laravel 13, cercadinhos Docker por aluno, squads como matéria.
